# 🟠 Islam and Socialist Systems — الإسلام والمناهج الاشتراكية

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **الإسلام والمناهج الاشتراكية (Islam and Socialist Systems)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on Islamic economics vs. socialism, 6 comparative analysis sections, daily habit tracker, knowledge quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 themes, narrator, responsive design, PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.
